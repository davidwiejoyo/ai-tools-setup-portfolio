# AI Tools Setup Portfolio

## Overview
This project documents my process of setting up an AI-assisted development environment using Cursor IDE, Claude Code, Codex, and GitHub.

The goal of this task is to demonstrate my ability to follow instructions, explore new tools independently, and troubleshoot technical issues along the way.

---

## Tools Installed

- Cursor IDE
- Claude Code extension
- Codex extension
- Git (installed manually)
- GitHub

---

## Steps Completed

1. Installed Cursor IDE and completed onboarding.
2. Selected "By Myself" during setup to proceed as an individual user.
3. Connected my GitHub account via browser authentication.
4. Installed Claude Code extension.
5. Installed Codex extension and logged in using my ChatGPT account.
6. Created a public GitHub repository.
7. Installed Git after realizing it was required.
8. Cloned the GitHub repository to my local machine.
9. Opened the repository inside Cursor IDE.
10. Created this README.md file.
11. Configured Git username and email for committing changes.
12. Committed and pushed the project to GitHub.

---

## Issues Encountered & How I Solved Them

### 1. Git Not Installed
At first, I could not find the "Git: Clone" option inside Cursor.

**Solution:**  
I realized that Git was not installed on my system. I installed Git manually from the official website, restarted Cursor, and the Git features became available.

---

### 2. Confusion Between Local and Remote Repository
Initially, I was confused because Cursor was working with local folders, while the repository already existed on GitHub.

**Solution:**  
I learned that I needed to clone the GitHub repository into my local machine. After cloning and opening the correct folder in Cursor, the project was properly connected.

---

### 3. Git Configuration Required Before Commit
When attempting to commit changes, Git required me to configure a username and email.

**Solution:**  
I set up my Git identity using global configuration (`user.name` and `user.email`), which allowed me to proceed with commit and push successfully.

---

### 4. Claude Code Login Limitation
After installing Claude Code, I was prompted with login options that require a subscription or API access.

**Solution:**  
Since this task does not require full API integration, I chose not to proceed with authentication. Instead, I focused on completing the setup and understanding how the extension integrates within the environment.

---

### 5. Codex Authentication
During Codex setup, I was able to log in using my ChatGPT account.

This allowed me to explore how Codex works as an AI coding assistant within Cursor. I also noted that advanced usage may require API configuration or billing, but basic login was sufficient for this task.
